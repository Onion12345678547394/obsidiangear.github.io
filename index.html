Obsidian G. - Projektinformationen

Technische Dokumentation der E-Commerce Website

Projektübersicht

Obsidian G. ist eine moderne E-Commerce-Webseite für Technologieprodukte mit Fokus auf Gaming-Hardware und Mobilgeräte.

Zur Live-Website

Hauptmerkmale:

  • Responsive Design für alle Geräte
  • Dynamischer Warenkorb
  • Mehrere Produktkategorien
  • Kryptowährungszahlungen
  • FAQ-Bereich

Seitenstruktur

Hauptbereiche:

  1. Header mit Navigation
  2. Produktkategorien:
    • Consoles
    • PC/Notebook
    • Graphics Cards
    • Phones
    • Coming Soon
  3. Warenkorb (rechte Seitenleiste)
  4. FAQ-Bereich

Verwendete Technologien

HTML5 CSS3 JavaScript Responsive Design

Dateistruktur:

  • index.html - Hauptseite
  • index.css - Stylesheet
  • index.js - Hauptskript

Besondere Funktionen

Warenkorb-System

Dynamische Berechnung der Gesamtsumme und Checkout-Funktion.

Zahlungsmethoden

Unterstützung für Kryptowährungen:

  • Bitcoin
  • Monero
  • Ethereum
  • Litecoin

Produktgarantie

Alle Produkte (außer Nintendo DS-Serie) sind originalverpackt.
